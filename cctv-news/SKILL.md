# CCTV 新闻联播摘要

## 何时启用

用户需要「昨天」新闻联播文字版的**抓取 + 通读 + 结构化摘要**时使用本 Skill（非实时直播稿；以站点当日文字版为准）。

## 执行步骤

1. **确认基准日**：取得「今天」日期 `YYYY-MM-DD`（以用户环境或对话约定为准）。
2. **目标日**：昨天 = 今天减 1 天；下文记为 `目标日`（年 / 月 / 日用于 URL 与标题）。
3. **构建 URL**（见下节），**抓取**页面正文并**完整阅读**所有条目。
4. 对每条新闻提取：**时间、地点、人物、事件、关键数据**（能取则取）。
5. 按「输出格式」生成最终回复；遵守「总结原则」。

## URL 构建

- **站点**：`http://mrxwlb.com/`
- **路径模式**（昨天）：  
  `http://mrxwlb.com/YYYY/MM/DD/YYYY年MM月DD日新闻联播文字版/`  
  其中 `YYYY/MM/DD` 与路径段中的中文日期均为 **目标日**，月份、日期补零与站点一致（通常两位月、两位日）。
- **编码**：若需 percent-encoding，中文路径段按 UTF-8 编码（与站点实际链接一致即可）。

## 输出格式（必须遵守）

### 标题区

```markdown
━━━━━━━━━━━━━━━━━━━━
📺 【YYYY年M月D日 新闻联播摘要】
━━━━━━━━━━━━━━━━━━━━
```

（`YYYY年M月D日` 与 **目标日** 一致。）

### 正文分区

每一区：emoji + **粗体标题**，接一行 `━━━━━━━━━━━━━━━━━━━━`，再列要点（`•`）。**区与区之间空一行**。

| 分区 | 标题 |
|------|------|
| 时政要闻 | 🔴 **时政要闻** |
| 国内新闻 | 🟠 **国内新闻** |
| 国际新闻 | 🟢 **国际新闻** |
| 数据与成就 | 📊 **重要数据 / 成就** |

### 排版规则

1. 标题块上下用 `━━━━━━━━━━━━━━━━━━━━` 分隔。
2. 要点统一用 `•`；重要数字、人名可 **加粗**。
3. 不省略该期节目中**头条级或会议/政策类**关键条目（见下节）。

## 总结原则

- **保留**：领导人活动与讲话要点；重大政策与法规；重要经济数据；重大科技与工程进展；外交与国际大事；民生政策与数据。
- **避免**：遗漏 obvious 头条或重要会议报道；编造未出现在正文中的事实或数字。
- **若页面不可用**（404、超时、正文过短）：说明失败原因与已尝试的 URL，不杜撰新闻内容。
